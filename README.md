# 🚀 Simulateur d'échéancier de prêt immobilier

Un outil complet et professionnel pour simuler des prêts bancaires avec gestion avancée des options.

## Fonctionnalités
- **Calcul d'échéancier** : Visualisation mois par mois du capital et des intérêts.
- **Gestion du différé** : Choix entre paiement des intérêts intercalaires ou capitalisation.
- **Tranches de taux** : Possibilité de définir plusieurs taux d'intérêt selon la période.
- **Assurance flexible** : Calcul au taux (initial/restant) ou montant fixe.
- **Remboursements anticipés** : Simulation d'impact sur la durée ou la mensualité.
- **Export & Import** : Génération de rapports PDF et sauvegarde/chargement via fichiers JSON.
- **Graphique dynamique** : Visualisation de l'amortissement avec Chart.js.
- **Hébergement en ligne** : Déployé via GitHub Pages.

## Installation
1. Clonez le dépôt.
2. Ouvrez `index.html` dans votre navigateur.
